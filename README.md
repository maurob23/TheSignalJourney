# The Signal Journey

**The Signal Journey** è un progetto editoriale, didattico e tecnico dedicato al viaggio del segnale: dalla vibrazione di una corda alla percezione della musica.

Il progetto integra:

- volumi editoriali;
- corso MusicJourney;
- progetti Ableton Live;
- Hardware Lab;
- simulazioni;
- documentazione tecnica;
- idee fondanti;
- repository GitHub.

## Principio guida

Il protagonista dell'opera non è soltanto la chitarra, l'amplificatore o il sintetizzatore.

Il protagonista è **il viaggio del segnale**.

Durante questo viaggio il segnale trasporta informazione, viene sostenuto dall'energia e attraversa una lunga catena di trasformazioni fisiche, elettriche, digitali e percettive.
