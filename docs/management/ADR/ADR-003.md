# ADR-003 — Repository separata per The Signal Journey

## Stato

Accettata

## Contesto

The Signal Journey e MusicJourney sono collegati, ma hanno finalità diverse.

## Decisione

The Signal Journey avrà una repository dedicata, separata da MusicJourney.

## Conseguenze

La repository TheSignalJourney conterrà:

- testi dell'opera;
- documentazione;
- Hardware Lab;
- materiali Ableton collegati al libro;
- risorse editoriali.

La repository MusicJourney conterrà invece il software applicativo e il corso operativo.
