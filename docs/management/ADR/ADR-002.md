# ADR-002 — Markdown come formato sorgente

## Stato

Accettata

## Contesto

L'opera dovrà produrre testo editoriale, documentazione tecnica, esportazioni PDF/EPUB e contenuti per MusicJourney.

## Decisione

Il formato sorgente principale sarà Markdown.

## Conseguenze

I contenuti saranno:

- versionabili con Git;
- leggibili senza software proprietario;
- esportabili in altri formati;
- integrabili con strumenti automatici.
