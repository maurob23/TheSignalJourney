# The Signal Journey

## Project Context

### Versione 0.1

------------------------------------------------------------------------

# Visione del progetto

**The Signal Journey** è un ecosistema didattico che accompagna il
lettore alla scoperta del suono seguendo il percorso di un segnale,
dalla sua origine fino alla percezione da parte del cervello umano.

L'obiettivo è unire:

-   Musica
-   Chitarra
-   Teoria musicale
-   Sintesi sonora
-   Ableton Live
-   Analisi dei segnali
-   Elettronica analogica
-   DSP
-   Informatica
-   Intelligenza Artificiale

Il filo conduttore dell'intera opera è **il viaggio del segnale**.

# Obiettivo finale

Realizzare un'opera completa sul suono, capace di accompagnare il
lettore dalla musica fino alla progettazione di un moderno sistema di
elaborazione audio, mantenendo sempre il collegamento tra teoria,
pratica e sperimentazione.
